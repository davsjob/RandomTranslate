# RandomTranslate
A script that randomly translates text

Clone this repository into your folder
To create a `venv` with the name ``rndtrns`` run the following command:
```
python -m venv randomtranslate
```
Then activate it using:

**Windows**
```
cd /path/randomtranslate/rndtrns
scripts/activate.ps1
```
**INSTALL DEPENDENCIES**

`pip install -r requirements.txt`

Then run it through

```
python random_translate.py INPUTFILE.pdf OUTPUTFILE.pdf
```
